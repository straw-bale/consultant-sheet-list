How It Works

Drop a PDF — sheet numbers and titles are extracted client-side using PDF.js. No file is uploaded. Works on any size PDF.

Pin regions — use Sheet # region and Title region to draw exact boxes over the title block cells. Applied to every page. Without pins, the tool auto-scans the bottom-right corner first, then bottom-left, then the full page.

Discipline + sort order are derived from the sheet number prefix: S/SD/SS → Structural · C → Civil · P → Plumbing · M → Mechanical · E → Electrical · FP → Fire Protection · T → Technology

CSV columns: NUMBER · SHEET NAME · DISCIPLINE · ORDER-MAJOR · ORDER-MINOR

PyRevit Plugin — Import to Revit

Once installed, this PyRevit button lets you import sheets directly into your Revit model from a CSV. Run it from Revit and it will open this page automatically, wait for you to generate and copy your sheet list, then create or update the sheets in your model in one step.

What you need
PyRevit must already be installed. If you don't have it, download it at pyrevitlabs.io and follow their setup guide before continuing.

How to install
1. Click Download Plugin below — the zip will save to your Downloads folder.
2. Open the zip and copy the ARTools.extension folder inside it.
3. Open File Explorer and paste this path into the address bar:
   %APPDATA%\pyRevit\Extensions
4. Paste ARTools.extension into that folder. If it already exists, merge — don't replace.
5. Open Revit, go to the pyRevit tab in the ribbon, and click Reload.
6. A new studioOS panel will appear with the Import Consultant Sheet List button.

Custom sheet parameters
The script will also populate DISCIPLINE, ORDER-MAJOR, and ORDER-MINOR on each sheet if those parameters exist in your model. These are optional — sheets will still be created if the parameters aren't present.
