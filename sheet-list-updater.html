How It Works

Drop a PDF — sheet numbers and titles are extracted client-side using PDF.js. No file is uploaded. Works on any size PDF.

Pin regions — use Sheet # region and Title region to draw exact boxes over the title block cells. Applied to every page. Without pins, the tool auto-scans the bottom-right corner first, then bottom-left, then the full page.

Discipline + sort order are derived from the sheet number prefix: S/SD/SS → Structural · C → Civil · P → Plumbing · M → Mechanical · E → Electrical · FP → Fire Protection · T → Technology

CSV columns: NUMBER · SHEET NAME · DISCIPLINE · ORDER-MAJOR · ORDER-MINOR
