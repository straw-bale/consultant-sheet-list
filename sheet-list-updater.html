How It Works

Automatic extraction — the tool scans the bottom-right corner of each page (most common title block position), then bottom-left, then the full page as a fallback. No region selection is required. If any titles come out wrong, use the Refine region button to draw a crop box and re-parse.

Discipline + sort order are derived from the sheet number prefix: S/SD/SS → Structural · P → Plumbing · M → Mechanical · E → Electrical · FP → Fire Protection · T → Technology

CSV columns: NUMBER · SHEET NAME · DISCIPLINE · ORDER-MAJOR · ORDER-MINOR — ready to import with the pyRevit button.
